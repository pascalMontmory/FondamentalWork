## Status Discipline

Use three labels when summarizing results:

- `verified`: derivation or numerical check is present and reproducible.
- `conditional`: result follows from stated assumptions but those assumptions are not established.
- `conjectural`: interesting direction without sufficient proof or validation.

Path changes introduced by this namespace do not change the scientific status of any claim.
