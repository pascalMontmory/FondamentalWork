# Relations UV/IR, holographie et cellule GUP

Ce dossier prolonge les notes precedentes en separant trois niveaux :

- identites exactes entre echelles de Planck, Hubble et densite critique ;
- resultats connus de type borne de trou noir, thermodynamique d'horizon et
  energie noire holographique ;
- equations candidates reliant le regulateur GUP UV a une projection IR
  gravitationnelle.

Resultat central :

```math
\rho_{\rm DE}
=
\frac{3\Omega_\Lambda}{8\pi}\rho_P
\left(\frac{\ell_P}{L_H}\right)^2.
```

Equation candidate unificatrice :

```math
\rho_{\rm eff}(L)
=
\Omega_L
\min\left[
\frac{g\rho_P}{16\pi^2\beta_0^2},
\frac{3c^4}{8\pi G L^2}
\right].
```

Nouvelle relation de croisement :

```math
L_\times
=
\beta_0\sqrt{\frac{6\pi}{g}}\ell_P.
```

Pour `beta0` d'ordre 1, la branche gravitationnelle domine a toute echelle
macroscopique. Le point difficile n'est donc pas la petitesse `10^-122`, mais
la dynamique covariante qui selectionne la bonne longueur IR et donne une
equation d'etat proche de `w=-1`.
