# VDF

Workspace for mathematical foundations related to verifiable delay functions.

## Current Status

This directory is a namespace placeholder. No VDF result is marked as verified here until the corresponding definition, theorem, proof, and/or reproducible test is added.

## Expected Structure

- `definitions.md`: objects, notation, and assumptions.
- `verified-results.md`: proven or reproducibly checked statements only.
- `conjectures.md`: unverified ideas and research directions.
- `tests/` or `scripts/`: reproducible computational checks when relevant.

## Verification Rule

Do not move a statement from conjectural to verified unless the proof or check is local to this directory and can be independently inspected.
