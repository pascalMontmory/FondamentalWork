# Collatz-Montmory Computational Verification

Computational checks can support bounded claims only. They do not prove the unbounded Collatz conjecture.

## Required Metadata

Every verification run should record:

- exact command;
- code version or commit SHA;
- integer range checked;
- stopping condition;
- timeout or overflow behavior;
- expected output;
- machine-independent assumptions.

No Collatz-Montmory range verification is included in this clean restructuring branch. Add scripts only when the definitions are stable enough to test.
