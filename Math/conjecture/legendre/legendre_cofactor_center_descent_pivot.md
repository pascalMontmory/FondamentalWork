# Cofactor-Center Descent Pivot

This note records a genuine descent structure in the shared-\(5\) certificate.

It is not a proof of Legendre.  It is a new proof skeleton: every
shared-\(5\) certificate vertex either descends to a smaller center, or lies
in a terminal cone where its prime label is forced below
\[
  (\sqrt2-1)A.
\]

This is different from killing energy layers one by one.

## 1. Symmetric cofactor variables

In the shared-\(5\) fiber, every variable offset
\[
  c\in\mathcal C=\{5,17,25,49,65\}
\]
has a centered certificate
\[
  (A-r_c)e_c=r_c^2+c.
\]

Since \(A\) is even and every \(c\equiv1\pmod4\), the previous mod-\(4\)
gate gives
\[
  e_c\equiv2\pmod4.
\]

Write
\[
  e_c=2u_c,
  \qquad u_c\text{ odd}.
\]

Also write the prime label
\[
  p_c=A-r_c.
\]

Then the certificate equation becomes
\[
\boxed{
  r_c^2+c=2u_cp_c.
}
\]

Now define the cofactor-center
\[
\boxed{
  W_c=r_c+u_c.
}
\]

## 2. Descent identity

The key identity is
\[
\boxed{
  W_c^2+c=u_c(2A+u_c).
}
\]

Indeed,
\[
\begin{aligned}
W_c^2+c
&=(r_c+u_c)^2+c\\
&=r_c^2+c+2u_cr_c+u_c^2\\
&=2u_cp_c+2u_cr_c+u_c^2\\
&=u_c(2(p_c+r_c)+u_c)\\
&=u_c(2A+u_c).
\end{aligned}
\]

Also
\[
\boxed{
  A-W_c=p_c-u_c.
}
\]

Therefore:

- if
  \[
    u_c<p_c,
  \]
  then
  \[
    W_c<A;
  \]
- if
  \[
    u_c\ge p_c,
  \]
  then
  \[
    W_c\ge A.
  \]

Thus every certificate vertex is either descending or terminal.

## 3. Meaning of descent

If \(u_c>1\), every prime divisor
\[
  \ell\mid u_c
\]
satisfies
\[
  \ell\mid W_c^2+c.
\]

If \(u_c<p_c\), then \(W_c<A\), so the original certificate at center \(A\)
has produced a smaller-center divisibility event
\[
\boxed{
  \ell\mid W_c^2+c,
  \qquad W_c<A.
}
\]

This is the first exact descent map found in the shared-\(5\) route:
\[
\boxed{
  (A,c,p_c,u_c,r_c)
  \longmapsto
  (W_c,c,\ell),
  \qquad W_c=r_c+u_c<A.
}
\]

The map is not yet a proof of Legendre because it descends a vertex, not an
entire bad interval certificate.  But it gives a concrete discrete-Morse
candidate: match every top cell containing a descending vertex to the lower
center event generated by that vertex.

The only possible critical cells are those where every vertex is terminal.

## 4. Terminal cone

A vertex is terminal exactly when
\[
  u_c\ge p_c.
\]

Since
\[
  r_c^2+c=2u_cp_c,
\]
terminality implies
\[
  r_c^2+c\ge2p_c^2.
\]

Substitute
\[
  r_c=A-p_c.
\]

Then
\[
  (A-p_c)^2+c\ge2p_c^2.
\]

Equivalently,
\[
  p_c^2+2Ap_c-A^2-c\le0.
\]

Solving the quadratic inequality gives the terminal label bound
\[
\boxed{
  p_c
  \le
  -A+\sqrt{2A^2+c}.
}
\]

Since \(c\le65\),
\[
\boxed{
  p_c
  \le
  -A+\sqrt{2A^2+65}
  =
  (\sqrt2-1)A+O(A^{-1}).
}
\]

Thus a terminal shared-\(5\) top cell must use five distinct prime labels all
lying far below the center:
\[
\boxed{
  p_c\le(\sqrt2-1)A+O(A^{-1})
  \qquad(c\in\mathcal C).
}
\]

This is a much stronger qualitative condition than the previous rank
barriers, which only forced one label below \(A-O(\sqrt A)\).

## 5. Dichotomy for shared-\(5\)

Every large shared-\(5\) compressed obstruction satisfies exactly one of the
following alternatives.

### Descent alternative

At least one offset has
\[
  u_c<p_c.
\]

Then the obstruction contains a canonical smaller-center divisibility event
\[
  \ell\mid W_c^2+c,
  \qquad W_c<A,
\]
where
\[
  W_c=r_c+u_c,
  \qquad \ell\mid u_c.
\]

This is the discrete-Morse direction.

### Terminal alternative

All five offsets satisfy
\[
  u_c\ge p_c.
\]

Then all five prime labels satisfy
\[
  p_c\le -A+\sqrt{2A^2+c}
  \le -A+\sqrt{2A^2+65}.
\]

Thus a terminal top cell is forced into the low-label cone
\[
  p_c\lesssim0.4143A.
\]

This is a narrow global incidence problem, not an energy-layer problem.

## 6. Why this is a real pivot

The previous route was:
\[
  \mathcal E=50,54,58,62,66,70,74,78,\ldots
\]
and could continue indefinitely.

The cofactor-center identity replaces that with:
\[
\boxed{
  \text{top cell}
  \Rightarrow
  \text{descent}
  \quad\text{or}\quad
  \text{all-terminal low-label cone}.
}
\]

So the next proof target is no longer an energy layer.  It is:

> Prove that all-terminal shared-\(5\) top cells are impossible, and prove
> that a descending vertex can be used as an acyclic Morse matching on the
> global certificate complex.

If both statements are proved, the shared-\(5\) channel closes without
climbing energy layers.

## 7. Exact lemmas now needed

The pivot reduces the elegant closure to two precise lemmas.

### Lemma D: Morse descent lemma

Every shared-\(5\) top cell containing a descending vertex
\[
  u_c<p_c
\]
is matched acyclically to a smaller-center cell through
\[
  W_c=r_c+u_c.
\]

This would turn the identity
\[
  W_c^2+c=u_c(2A+u_c)
\]
into a genuine topological descent.

### Lemma T: terminal cone non-realization

There is no large shared-\(5\) top cell with
\[
  u_c\ge p_c
  \qquad(c\in\{5,17,25,49,65\}).
\]

Equivalently, there is no shared-\(5\) top cell whose five labels all satisfy
\[
  p_c\le -A+\sqrt{2A^2+c}.
\]

This is the new hard target.  It is finite-dimensional, geometric, and not
an infinite energy ladder.

## 8. First terminal equations

In terminal variables, write
\[
  u_c=p_c+h_c,
  \qquad h_c\ge0.
\]

Then
\[
  r_c=A-p_c,
\]
and
\[
  (A-p_c)^2+c=2p_c(p_c+h_c).
\]

Equivalently,
\[
\boxed{
  A^2-2Ap_c-p_c^2+c=2p_ch_c.
}
\]

Thus every terminal vertex satisfies the divisibility condition
\[
\boxed{
  2p_c\mid A^2+c
}
\]
with the stronger inequality
\[
  p_c\le -A+\sqrt{2A^2+c}.
\]

The all-terminal cone is therefore the incidence system:
\[
\begin{cases}
p_c\mid A^2+c,\\
p_c\le -A+\sqrt{2A^2+c},\\
p_c\ne p_d\quad(c\ne d),\\
A\equiv12,18\pmod {30},\\
c\in\{5,17,25,49,65\}.
\end{cases}
\]

This is now the correct object to attack.

## 9. Euclidean descent inside the terminal cone

The terminal cone is not a dead end.  It has its own Euclidean descent.

Let a terminal label satisfy
\[
  p\mid A^2+c,
  \qquad
  p\le -A+\sqrt{2A^2+c}.
\]

Divide \(A\) by \(p\):
\[
\boxed{
  A=kp+s,\qquad 0\le s<p.
}
\]

Since
\[
  A^2+c\equiv s^2+c\pmod p,
\]
we get
\[
\boxed{
  p\mid s^2+c.
}
\]

Write
\[
\boxed{
  s^2+c=vp.
}
\]

This gives a smaller-center object because
\[
  s<A.
\]

If
\[
  1<v<s,
\]
then some prime
\[
  \ell\mid v
\]
satisfies
\[
  \ell\le v<s,
\]
and therefore
\[
\boxed{
  \ell\mid s^2+c,\qquad \ell<s.
}
\]

So the terminal label has descended to a genuine smaller-center
divisibility event at center \(s\).

The only terminal atoms that do not immediately descend are:

1. the prime-value atom
   \[
     v=1,\qquad p=s^2+c;
   \]
2. finite small cases where \(v\ge s\), equivalently
   \[
     s^2+c\ge sp.
   \]

For large \(s>c\), the second case is essentially absent because \(p>s\)
implies
\[
  v=\frac{s^2+c}{p}<s+\frac cp<s+1,
\]
so \(v\le s\), and \(v=s\) forces
\[
  p=s+\frac cs,
\]
hence \(s\mid c\).

Thus, outside finitely many small divisors of \(c\), terminal labels also
descend unless they are prime-value atoms
\[
\boxed{
  p=s^2+c.
}
\]

## 10. Terminal quotient split

The Euclidean quotient \(k\) also sharpens the geometry.

The terminal bound
\[
  p\le(\sqrt2-1)A+O(A^{-1})
\]
implies
\[
  \frac Ap\ge1+\sqrt2+O(A^{-2}).
\]

Therefore
\[
  k\ge2.
\]

If
\[
  p>\frac A3,
\]
then
\[
  k=2
\]
and
\[
  s=A-2p.
\]

In that boundary-terminal strip,
\[
  p\le -A+\sqrt{2A^2+c}
\]
is equivalent to
\[
  s\ge(\sqrt2-1)p+O(p^{-1}).
\]

So the hardest terminal atoms live in the explicit strip
\[
\boxed{
  A=2p+s,\qquad
  (\sqrt2-1)p\lesssim s<p,\qquad
  p\mid s^2+c.
}
\]

Then either:

- \(s^2+c\) is composite with a prime factor below \(s\), giving descent;
- or
  \[
    p=s^2+c
  \]
  and the terminal label is generated by a prime value of the same offset at
  the smaller center \(s\).

## 11. Refined closure target

The shared-\(5\) route is no longer an infinite energy ladder.

Every vertex now falls into one of three exact types:

1. **cofactor descent**
   \[
     u<p,\qquad W=r+u<A;
   \]
2. **terminal Euclidean descent**
   \[
     u\ge p,\quad A=kp+s,\quad s^2+c=vp,\quad 1<v<s;
   \]
3. **terminal prime atom**
   \[
     u\ge p,\quad A=kp+s,\quad p=s^2+c
   \]
   up to finite small divisor exceptions.

Thus the real elegant proof target is:

> Show that a full shared-\(5\) top cell cannot consist entirely of terminal
> prime atoms and finite small exceptions, and construct an acyclic matching
> for the two descent types.

This is the first route that could close shared-\(5\) without raising the
energy threshold indefinitely.
